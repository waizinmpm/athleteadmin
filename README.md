# Oversea-admin

## Project setup
```
npm install
```

- Create ```.env``` file in project root folder.
- In ```.env``` file, write
- ```VUE_APP_API_URL=http://your-oversea-jobs-project-run-ip/api/```
- Be sure your ip address.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
